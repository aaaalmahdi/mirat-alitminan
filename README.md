# مرآة الاطمئنان

تطبيق ويب ثابت مخصص للبوث. يفتح عبر QR، يطلب موافقة المستخدم، يشغل الكاميرا داخل إطار قلب، يقدر العمر الظاهري كنطاق ترفيهي، ثم يعرض الباقة الوقائية المناسبة حسب العمر الحقيقي والجنس.

## الخصائص

- واجهة عربية RTL موبايل.
- بدون Backend وبدون قاعدة بيانات.
- لا حفظ للصور أو الفيديو.
- مسار بديل بدون كاميرا.
- منطق الباقات حسب العمر والجنس.
- جاهز للنشر على GitHub Pages أو Vercel أو Netlify.

## تشغيل محلي

```bash
python3 -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080
```

ملاحظة: الكاميرا على الجوال تحتاج HTTPS عند النشر.

## تفعيل تقدير العمر الحقيقي

الكود يحاول تحميل `face-api.js` من CDN، ثم يبحث عن ملفات النماذج داخل:

```text
models/
```

ضع ملفات النماذج التالية داخل مجلد `models`:

```text
tiny_face_detector_model-weights_manifest.json
tiny_face_detector_model-shard1
age_gender_model-weights_manifest.json
age_gender_model-shard1
```

إذا لم تكن النماذج موجودة، سيعمل التطبيق بالمسار البديل حتى لا تتعطل تجربة البوث.

## النشر على GitHub Pages

1. أنشئ repository جديدًا.
2. ارفع هذه الملفات إلى الجذر.
3. من Settings > Pages اختر Deploy from branch.
4. اختر branch `main` والمجلد `/root`.
5. استخدم رابط HTTPS الناتج لتوليد QR.

## النشر على Vercel أو Netlify

ارفع المجلد كما هو. لا يحتاج Build Command.

إعدادات مقترحة:

```text
Build command: empty
Output directory: .
```
