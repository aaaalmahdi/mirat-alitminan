# مرآة الاطمئنان

تطبيق ويب ثابت للبوث الصحي. يفتح عبر QR، يحدد الباقة الوقائية حسب العمر الحقيقي والجنس، ويضيف مسار كاميرا اختياري للعمر الظاهري كتجربة توعوية فقط.

## نسخة V4 UX

- شاشة أولى مبسطة: ابدأ الآن + معلومات عن الخدمة.
- شاشة اختيار مسار: معرفة الباقة مباشرة أو تجربة العمر الظاهري أولاً.
- أسئلة اختيارية قصيرة قبل النتيجة:
  - سكري/ضغط/دهون.
  - فحص وقائي خلال آخر سنة.
  - تاريخ عائلي لأمراض مزمنة.
- نتيجة أوضح لكل باقة:
  - اسم الباقة.
  - العمر المستهدف.
  - الفحوصات المقترحة.
  - ماذا يعني ذلك لك؟
  - ملاحظة للموظف.
  - مساعد توعوي آمن لا يشخص.
- بطاقة موظف مختصرة مع ملاحظات المستخدم الاختيارية.
- شاشة شكر بعد الانتهاء.
- صفحة معلومات إضافية مستقلة `info.html`.
- لوحة قياس محلية أوسع:
  - الزيارات.
  - بدء التجربة.
  - المسار المباشر.
  - مسار الكاميرا.
  - فتح الكاميرا.
  - فشل الكاميرا.
  - نتائج الباقات.
  - بطاقات الموظف.
  - فتح صفحة المعلومات.
  - نسبة الإكمال.
- نسخة احتياطية داخل `docs/` في حال تم ضبط GitHub Pages على مجلد docs.

## الخصوصية

لا يتم حفظ الصور أو الفيديو. الإحصاءات تحفظ محلياً على جهاز البوث فقط، ولا تتضمن أسماء أو أرقام هواتف أو أعماراً فردية.

## تعديل الباقات

عدّل ملف:

```text
packages.json
```

يمكن تغيير:

```text
code, min, max, gender, color, name, range, tags, reason, meaning, nextStep, checks
```

## لوحة الإحصاءات

اضغط على منطقة أعلى اليسار 5 مرات متتالية لفتح إحصاءات الجهاز.

## تشغيل محلي

```bash
python3 -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080
```

الكاميرا على الجوال تحتاج HTTPS عند النشر.
