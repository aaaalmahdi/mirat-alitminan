# مرآة الاطمئنان

تطبيق ويب ثابت للبوث الصحي. يفتح عبر QR، يحدد الباقة الوقائية حسب العمر الحقيقي والجنس، ويضيف مسار كاميرا اختياري للعمر الظاهري كتجربة توعوية فقط.

## نسخة V5 Reliability

- أضيف fallback كامل للباقات داخل `app.js` حتى لا يتعطل البوث إذا فشل تحميل `packages.json`.
- انتقلت قواعد الفحوصات الشرطية إلى `packages.json` في `conditionalChecks`.
- أضيف نص خصوصية واضح أسفل الأسئلة الاختيارية.
- صفحة المعلومات تعرض الآن: ما هي الخدمة، الرؤية، الرسالة، القيم الأساسية، الباقات، الاستحقاق، والأسئلة الشائعة.
- مسار النشر الموصى به أصبح واحداً: GitHub Actions.

## الخصوصية

لا يتم حفظ الصور أو الفيديو. الإحصاءات تحفظ محلياً على جهاز البوث فقط، ولا تتضمن أسماء أو أرقام هواتف أو أعماراً فردية. إجابات الأسئلة الاختيارية لا تحفظ في التحليلات، وتستخدم فقط لإظهار ملاحظة للمستخدم والموظف داخل نفس الجلسة.

## تعديل الباقات

عدّل ملف:

```text
packages.json
```

يمكن تغيير:

```text
code, min, max, gender, color, name, range, tags, reason, meaning, nextStep, checks, conditionalChecks
```

## لوحة الإحصاءات

اضغط على منطقة أعلى اليسار 5 مرات متتالية لفتح إحصاءات الجهاز.

## تشغيل محلي

```bash
python3 -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080
```

الكاميرا على الجوال تحتاج HTTPS عند النشر.
