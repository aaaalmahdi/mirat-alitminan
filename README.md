# مرآة الاطمئنان

تطبيق ويب ثابت للبوث الصحي. يفتح عبر QR، يعطي تجربة اختيارية للعمر الظاهري، ثم يعرض الباقة الوقائية والفحوصات المقترحة حسب العمر الحقيقي والجنس.

## نسخة V2 Booth Ready

- تصميم موبايل بدون إطار هاتف أسود.
- مسار مباشر لمعرفة الباقة بدون كاميرا.
- تجربة كاميرا اختيارية فقط.
- إيقاف الكاميرا بعد ظهور النتيجة.
- تقدير العمر كنطاق ترفيهي، وليس رقمًا طبيًا.
- ملف `packages.json` لتعديل الباقات والفحوصات بدون لمس الكود.
- بطاقة مختصرة للموظف.
- رجوع تلقائي للبداية بعد النتيجة.
- إحصاءات محلية بدون بيانات شخصية.
- دعم عربي/إنجليزي.
- نسخة احتياطية داخل `docs/` في حال تم ضبط GitHub Pages على مجلد docs.

## الخصوصية

لا يتم حفظ الصور أو الفيديو. الإحصاءات تحفظ محليًا على جهاز البوث فقط، ولا تتضمن أسماء أو أرقام هواتف أو أعمارًا فردية.

## تعديل الباقات

عدّل ملف:

```text
packages.json
```

يمكن تغيير:

```text
code, min, max, gender, color, name, range, tags, reason, checks
```

## لوحة الإحصاءات

اضغط على شعار/منطقة أعلى اليسار 5 مرات متتالية لفتح إحصاءات الجهاز.

## تشغيل محلي

```bash
python3 -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080
```

الكاميرا على الجوال تحتاج HTTPS عند النشر.

## GitHub Pages

الملفات جاهزة لثلاث حالات:

```text
main / root
main / docs
gh-pages / root
```

الرابط النهائي المتوقع:

```text
https://aaaalmahdi.github.io/mirat-alitminan/
```
